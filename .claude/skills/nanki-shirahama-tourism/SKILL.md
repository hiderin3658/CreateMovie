# 南紀白浜観光プロモーション動画スキル

白浜の実際の観光写真（16枚）を基に、「行ってみたい」と思わせる4カット構成の動画ストーリーを自動生成します。

## このスキルができること

✅ **実際の素材に基づいたストーリー生成**
- 16枚の実写真を活用（白良浜、円月島、三段壁、崎の湯、千畳敷、番所山隧道、熊野古道）
- Claude解析済みの詳細メタデータを参照
- 存在しない場所（アドベンチャーワールドなど）を除外

✅ **🆕 Deep Researchデータベース統合**
- 10箇所の観光地の詳細分析データを活用
- 各ロケーションのコアナラティブ、ストーリーテーマ
- 最適な撮影時間、構図、音響設計の提案
- 3種類のストーリーフレームワーク（時間、対比、五感）

✅ **4カット構成（各10秒）**
- 4カット × 10秒 = 40秒コンテンツ（編集後約30秒）
- SNS向け（TikTok, Instagram Reels, YouTube Shorts）
- ストーリーアーク: 出会い → 自然の驚き → 体験の楽しみ → もう一度来たい

✅ **観光プロモーション最適化**
- 白浜の魅力を効果的に訴求
- カテゴリ別素材配分（beach, nature, attractions, culture）
- 提供写真のモチーフ変更禁止（コンペ制約遵守）
- リサーチデータに基づく高精度な素材マッチング

## 使い方

### 基本的な使い方

```
白浜プロジェクトで観光に行ってみたいと思わせる動画のストーリーを4カット構成で5つ考えて
```

自動的に：
1. 実際の素材リスト（16枚）を読み込み
2. Claude解析メタデータを参照
3. 4カット構成で複数のストーリー案を生成
4. 各カットに実際の場所を割り当て

### より具体的な指定

```
白浜プロジェクトで夕日をテーマにした4カット構成の動画を作って
```

```
白浜プロジェクトで白良浜と円月島を中心にした4カット構成のストーリーを3つ考えて
```

### 🆕 ストーリーフレームワークを指定

Deep Researchに基づく3種類のフレームワークから選択できます：

**A案: 対比の物語**
```
白浜プロジェクトで「対比の物語」フレームワークを使って、
古代と現代の対比を表現する4カット動画を作って
```
- テーマ: 古代と現代、静と動、自然と人工
- ターゲット: 知的で物事の多面性を楽しむ層
- 例: 熊野古道の静 vs SUNTIDEの動

**B案: 時間の物語（最も推奨）**
```
白浜プロジェクトで「時間の物語」フレームワークを使って、
1500万年の地質から現代リゾートまでの時間軸で4カット動画を作って
```
- テーマ: 1500万年から15分まで、白浜で交錯する時間
- ターゲット: 歴史や地質学に興味があり、深い学びを旅に求める層
- タイムライン: 千畳敷(1500万年前) → 熊野古道(平安時代) → 三段壁(源平合戦) → 白良浜(現代)

**C案: 五感の物語**
```
白浜プロジェクトで「五感の物語」フレームワークを使って、
視覚・聴覚・触覚で白浜を体験する4カット動画を作って
```
- テーマ: 白浜で研ぎ澄まされる五感
- ターゲット: 癒しと感覚的な体験を最優先する層
- マッピング: 視覚→円月島/花火、聴覚→崎の湯/千畳敷、触覚→温泉/白良浜

### リサーチデータを活用した高度な指定

```
白浜プロジェクトで円月島のコアナラティブ「希少性という名のロマン」を
中心に据えた4カット動画を作って。春分・秋分期の奇跡の瞬間を強調して
```

```
白浜プロジェクトで崎の湯の「荒々しい自然との一体化」というテーマで、
波の轟音と湯の流れのコントラストを表現する動画を作って
```

## 利用可能な素材（16枚）

### Beach（5枚）
- **白良浜**: 白い砂浜、エメラルドグリーンの海、リゾートホテル（空撮、高台、ビーチレベル）
- **ビーチカフェ**: モダンな白い建物

### Nature（6枚）
- **三段壁**: 50m断崖絶壁、層状の岩
- **円月島**: 穴の開いた象徴的な岩の島（日中＋夕日シルエット）
- **千畳敷**: 波の浸食で作られた平らな岩の地形
- **番所山隧道**: 歴史的なレンガのトンネル

### Attractions（4枚）
- **崎の湯**: 海辺の露天風呂、ダイナミックな波
- **白良浜花火**: 夜の花火大会

### Culture（1枚）
- **熊野古道富田坂**: 世界遺産、石段の巡礼路


### 🆕 リサーチデータベースシステム

**ResearchDatabase クラス:**
- YAMLから構造化されたロケーションデータを読み込み
- シーン説明からロケーション提案（スコアリング）
- ストーリーフレームワークに基づく構造生成

**ResearchAwareStrategy クラス:**
- 通常のTourismStrategyを拡張
- リサーチデータに基づいて素材マッチング精度を向上
- コアナラティブ、ビジュアル要素、撮影時間などでボーナススコア付与

**スコアリングボーナス:**
- コアナラティブマッチ: +8点
- ビジュアル要素マッチ: +4点/単語
- 推奨撮影時間一致: +12点
- 優先ロケーション: +10点
- リサーチ提案トップ3: +20/+10/+7点

### 実行スクリプト
```bash
cd projects/nanki-shirahama-2024
python generate_tourism_videos_v2.py "ストーリーの説明"
```

- **ツール**: Gemini 2.5 Flash Image
- **参照画像**: 最大3枚まで使用可能
- **コスト**: $0.039/画像

- **推奨**: Sora 2（$0.10/秒、720p）

- **総コスト**: 約$4.16-7（4カット分）

## 🆕 Deep Researchデータベース

10箇所の観光地について、詳細なリサーチデータを活用できます：

### 主要ロケーションのコアナラティブ

| ロケーション | コアナラティブ | ストーリーテーマ |
|------------|-------------|---------------|
| **崎の湯** | 荒々しい自然との一体化 | 自然への帰順 |
| **白良浜** | 人間の情熱が維持する完璧な美 | キュレーションされた楽園 |
| **円月島** | 希少性という名のロマン | 「奇跡」のブランディング |
| **三段壁** | 自然の巨大さと人間の小ささ | 畏怖と謙虚さ |
| **千畳敷** | 時間の可視化 | 地質学的ロマン |
| **熊野古道** | 巡礼という旅の形 | 霊的な旅 |
| **番所山隧道** | 忘れられた歴史 | 歴史的レイヤー |

### 撮影ガイド情報

各ロケーションには以下が含まれます：
- **最適撮影時間**: 崎の湯→早朝/荒天時、円月島→春分・秋分期
- **推奨ショットタイプ**: 三脚固定、動ショット、空撮など
- **音響設計**: 波の轟音、風の音、石段を登る音など
- **ビジュアル要素**: 湯気と潮風、層状の岩、穴に沈む夕日

### ナラティブフレーズライブラリ

事前に作成されたフレーズを活用：

**オープニング:**
- "白浜には、四つの顔がある。古代の聖地、地質学の博物館、歴史的要衝、そして近代的なリゾート。"
- "1500万年の時間が、ここでは歩ける距離にある。"

**トランジション:**
- "目を閉じて、白浜の『音』を聞く。"
- "白浜の『上』には絶景がある。しかし、白浜の『下』には、物語がある。"

**クロージング:**
- "白浜。ここは、自然と人間の情熱が描き上げた、一枚の絵画である。"
- "最高の景色は、時として、歴史の影に隠されている。"

## Claude解析メタデータ活用

各素材には以下の詳細情報が記録されています：

```yaml
- filename: 円月島夕日１.jpeg
  location: 円月島
  description: 円月島の夕景。夕日が島の穴を通過するシルエット、
               オレンジ・ピンク・金色のグラデーション空、海面の反射。
  time_of_day: sunset
  weather: clear
  main_subject: 円月島と夕日
  composition: silhouette
  color_tone: orange_pink_gold
```

## 制約事項

### 素材制約（コンペルール）
- ⚠️ 提供写真のモチーフ（被写体）変更**禁止**
- ✅ 写真の拡大・縮小・トリミング**許可**
- ✅ アニメスタイルへの変換**許可**（構図保持）
- ✅ キャラクターの追加**許可**（背景として使用）

### カメラワーク制約
- ✅ 使用可能: Pan, Zoom, Tilt
- ⚠️ 禁止: 素材の形状を変形させるモーション

## 出力形式

### ストーリー構成
各動画（4カット × 10秒）：
1. **出会いの予感** (beach): 白良浜の美しさ
2. **自然の驚き** (nature): 円月島、三段壁、千畳敷
3. **体験の楽しみ** (attractions/culture): 崎の湯、熊野古道
4. **もう一度来たい** (beach/nature): 印象的シーン再登場

## トラブルシューティング

### 存在しない場所が生成される
→ このスキルは実際の素材リストをプロンプトに含めるため、存在しない場所（アドベンチャーワールドなど）の生成を自動的に防ぎます。

### カット数が4でない
→ プロンプトに「4カット構成」を明記してください。自動検出されます。

### 素材が見つからない
→ `source_materials/raw/` に16枚の写真が配置されているか確認してください。
